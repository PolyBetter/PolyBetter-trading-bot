## 📋 ДЕТАЛЬНОЕ ОПИСАНИЕ

### 1️⃣ `flexer78_all` - Агрессивная стратегия
**ТЕКУЩИЙ ПРЕСЕТ В ВАШЕМ БОТЕ**

```json
{
  "max_tick": 0.03,           // Принимает ставки до 3¢
  "bet_both_sides": true,     // ⚠️ СТАВИТ НА YES И NO ОДНОВРЕМЕННО
  "batch_size": 300,          // Обрабатывает 300 рынков за раз
  "prioritize_skewed": true,  // Приоритет перекошенным рынкам
  "min_hours_to_end": 3       // Минимум 3 часа до закрытия
}
```

**Особенности:**
- ✅ Ставит на ОБА исхода (YES + NO), если спред > 10%
- ✅ Принимает более дорогие тики (до 3¢)
- ✅ Большой batch = больше ордеров за скан
- ⚠️ **РЕЗУЛЬТАТ: В 2-3 РАЗА БОЛЬШЕ ОРДЕРОВ, чем в старой версии**

**Когда использовать:**
- Большой баланс (>$200)
- Хотите максимальный охват рынков
- Не боитесь двойных ставок на один рынок

---

### 2️⃣ `flexer78_safe` - Сбалансированная стратегия ⭐ РЕКОМЕНДУЮ
**МАКСИМАЛЬНО БЛИЗКО К СТАРОЙ ВЕРСИИ**

```json
{
  "max_tick": 0.01,           // Только до 1¢ (как в старой версии)
  "bet_both_sides": false,    // ✅ ТОЛЬКО 1 СТАВКА НА РЫНОК
  "batch_size": 100,          // Как в старой версии
  "prioritize_skewed": false, // НЕ приоритизирует перекосы
  "min_hours_to_end": 3       // 3 часа (как flexer78)
}
```

**Особенности:**
- ✅ БЕЗ двойных ставок (YES и NO)
- ✅ Только дешёвые тики ≤ 1¢
- ✅ Умеренный batch size
- ✅ **РАБОТАЕТ КАК СТАРАЯ ВЕРСИЯ, но с новыми фичами**

**Когда использовать:**
- Средний баланс ($50-200)
- Хотите контролируемое количество ордеров
- Предпочитаете консерватизм

---

### 3️⃣ `flexer78_ultra_safe` - Ультра-консервативная стратегия
**ТОЛЬКО САМЫЕ ДЕШЁВЫЕ СТАВКИ**

```json
{
  "max_tick": 0.001,          // ⚠️ ТОЛЬКО 0.1¢ ставки!
  "bet_both_sides": false,    // Только 1 ставка
  "batch_size": 100,
  "prioritize_skewed": false,
  "min_hours_to_end": 24,     // Минимум 1 день до закрытия
  "multi_outcome_threshold": 0.3  // Более строгий фильтр
}
```

**Особенности:**
- ✅ ТОЛЬКО 0.1¢ ставки (максимальный мультипликатор x1000)
- ✅ Минимум 1 день до закрытия рынка
- ✅ Более строгая фильтрация
- ✅ **МИНИМУМ ОРДЕРОВ, МАКСИМУМ БЕЗОПАСНОСТИ**

**Когда использовать:**
- Малый баланс (<$50)
- Хотите только "лотерейные" ставки
- Готовы ждать редких, но выгодных возможностей

## ⚠️ ВАЖНО!

### Почему старая версия "безопаснее":
```python
# СТАРАЯ ВЕРСИЯ:
token = tokens[0]  # Берёт ТОЛЬКО первый токен (YES)
# → 1 ставка на рынок

# НОВАЯ ВЕРСИЯ (flexer78_all):
if bet_both_sides and spread > 10%:
    # Ставит на ОБА токена (YES и NO)
    # → 2 ставки на рынок
```

### Чтобы новая работала как старая:
✅ Используйте пресет **`flexer78_safe`** или **`flexer78_ultra_safe`**

## 🔧 НАСТРОЙКА

Если хотите изменить параметры:
1. Откройте `presets.json`
2. Найдите нужный пресет
3. Измените параметры:
   - `max_tick` - максимальный тик (0.001, 0.01, 0.03)
   - `bet_both_sides` - ставить на оба исхода (true/false)
   - `batch_size` - размер батча (50-300)
   - `min_hours_to_end` - минимум часов до конца (1-48)
